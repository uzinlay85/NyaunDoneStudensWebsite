အလွန်ကောင်းမွန်တဲ့ စိတ်ကူးပါပဲခင်ဗျာ။ အခုလို စနစ်တကျ မှတ်တမ်း (Documentation) ပြန်လည်ရေးသားထားခြင်းက နောင်နှစ်တွေမှာ ဒီ Website ကို ပြန်လည်အသုံးပြုမယ့် ဒါမှမဟုတ် ဆက်လက်ထိန်းသိမ်းမယ့် ဘယ်သူ့အတွက်မဆို အလွန်တန်ဖိုးရှိပြီး အချိန်ကုန်သက်သာစေမှာ အသေအချာပါပဲ။

သင်ရေးသားထားတဲ့ မူရင်း README ဖိုင်က အလွန်ပြည့်စုံနေပါပြီ။ အခုနောက်ဆုံးပြောင်းလဲသွားတဲ့ **"ခုနှစ်အလိုက် စာရင်းပြောင်းလဲကြည့်ရှုနိုင်တဲ့ စနစ်"** နဲ့ ကိုက်ညီအောင် အချက်အလက်တွေကို ပိုမိုတိကျသွားအောင်နဲ့ နောင်နှစ်အတွက် အသုံးပြုပုံကို ပိုရှင်းလင်းသွားအောင် ကျွန်တော် ပြန်လည်မွမ်းမံတည်းဖြတ်ပေးလိုက်ပါတယ်ခင်ဗျာ။

### **ညောင်တုန်းကျောင်းတိုက် သံဃာတော်များ စာရင်း Website - နည်းပညာမှတ်တမ်းနှင့် အသုံးပြုပုံလမ်းညွှန် (ဗားရှင်း ၂.၀)**

ဒီမှတ်တမ်းမှာ အဓိကအားဖြင့် အပိုင်း (၄) ပိုင်း ပါဝင်ပါတယ်-

1.  **ခြုံငုံသုံးသပ်ချက် (Overview):** ဒီစနစ်က ဘာတွေလုပ်ပေးသလဲ။
2.  **ဖိုင်ဖွဲ့စည်းပုံ (File Structure):** ဘယ်ဖိုင်တွေပါဝင်ပြီး ဘယ်လိုချိတ်ဆက်ထားသလဲ။
3.  **အဓိကလုပ်ဆောင်ချက်များ (Key Features):** Website မှာပါဝင်တဲ့ အဓိက Feature တွေက ဘာတွေလဲ၊ ဘယ်လိုအလုပ်လုပ်သလဲ။
4.  **နောင်နှစ်အတွက် အချက်အလက်အသစ် ထည့်သွင်းနည်း (Step-by-Step Guide for Next Year):** နောက်နှစ်စာရင်း ပြောင်းတဲ့အခါ ဘာတွေလုပ်ရမလဲ။

---

### **၁။ ခြုံငုံသုံးသပ်ချက် (Overview)**

ဒီ Website ဟာ Server (ဒေတာဘေ့စ်) မလိုအပ်ဘဲ၊ ရိုးရှင်းတဲ့ ဖိုင်အနည်းငယ်နဲ့ သံဃာတော်များရဲ့ စာရင်းကို **ခုနှစ်အလိုက်** အပြန်အလှန်တုံ့ပြန်မှုရှိရှိ (Interactive) ကြည့်ရှုနိုင်၊ စီစစ်နိုင် (Filter)၊ ရှာဖွေနိုင်၊ နှင့် စီမံခန့်ခွဲနိုင်အောင် တည်ဆောက်ထားတဲ့ "Client-Side Web Application" တစ်ခုဖြစ်ပါတယ်။ အဓိက အချက်အလက်တွေကို ခုနှစ်အလိုက် သီးခြားစီခွဲထားတဲ့ CSV ဖိုင်များ (ဥပမာ: `students-2025.csv`, `students-2026.csv`) ထဲမှာ သိမ်းဆည်းပြီး၊ `index.html` ဖိုင်က အသုံးပြုသူရွေးချယ်တဲ့ ခုနှစ်အလိုက် သက်ဆိုင်ရာ CSV ဖိုင်ကို ခေါ်ယူဖတ်ရှုကာ Website အဖြစ် ပြောင်းလဲပေးပါတယ်။

---

### **၂။ ဖိုင်ဖွဲ့စည်းပုံ (File Structure)**

ဒီစနစ်တစ်ခုလုံး အလုပ်လုပ်ဖို့အတွက် အဓိကဖိုင်အမျိုးအစား (၃) မျိုး လိုအပ်ပါတယ်-

*   **`index.html`**
    *   **အမျိုးအစား:** ပင်မ Website ဖိုင်။
    *   **หน้าที่:** Website ရဲ့ မြင်တွေ့ရတဲ့ အပိုင်း (UI) အားလုံးနှင့် အလုပ်လုပ်ပုံ منطق (Logic) အားလုံးကို ထိန်းချုပ်တဲ့ JavaScript ကုဒ်တွေ ပါဝင်ပါတယ်။
    *   **ချိတ်ဆက်မှု:** အသုံးပြုသူရွေးချယ်လိုက်တဲ့ ခုနှစ် (`2025`, `2026`) ပေါ်မူတည်ပြီး သက်ဆိုင်ရာ CSV ဖိုင် (`students-2025.csv`, `students-2026.csv`) ကို `fetch` command သုံးပြီး ခေါ်ယူဖတ်ရှုပါတယ်။ `logo.png` ကို Website အမှတ်တံဆိပ်အဖြစ် အသုံးပြုပါတယ်။

*   **`students-YYYY.csv` (ဥပမာ: `students-2025.csv`)**
    *   **အမျိုးအစား:** ခုနှစ်အလိုက် အချက်အလက် (Data) ဖိုင်။
    *   **หน้าที่:** သက်ဆိုင်ရာခုနှစ်ရဲ့ သံဃာတော်များ အချက်အလက်အားလုံးကို ဇယားကွက်ပုံစံနဲ့ သိမ်းဆည်းထားတဲ့ "ဒေတာဘေ့စ်" ဖြစ်ပါတယ်။ နာမည်၊ အတန်း၊ ကျောင်း၊ ခုံနံပါတ်၊ ဘာသာရပ်၊ နှင့် အခြေအနေ (Status) တို့ ပါဝင်ပါတယ်။
    *   **အရေးကြီးသောအချက်:** Website ပေါ်မှာပြသတဲ့ အချက်အလက်အားလုံးရဲ့ **ပင်မရင်းမြစ် (Single Source of Truth)** ဖြစ်ပါတယ်။ ဒီဖိုင်ကို ပြင်ဆင်လိုက်တာနဲ့ Website က အလိုအလျောက် ပြောင်းလဲသွားပါမယ်။

*   **`logo.png`**
    *   **အမျိုးအစား:** ရုပ်ပုံဖိုင်။
    *   **หน้าที่:** Website ရဲ့ အမှတ်တံဆိပ် (Brand Identity) အတွက် Browser Tab နဲ့ ခေါင်းစဉ်တွေမှာ ပြသပါတယ်။

### **၃။ အဓိကလုပ်ဆောင်ချက်များ (Key Features)**

*   **ခုနှစ်အလိုက် စာရင်းပြောင်းလဲကြည့်ရှုခြင်း:**
    *   Website ရဲ့ထိပ်မှာရှိတဲ့ Dropdown မှတစ်ဆင့် ကြည့်ရှုလိုသော ခုနှစ် (၂၀၂၅၊ ၂၀၂၆) ကို ရွေးချယ်နိုင်ပြီး၊ သက်ဆိုင်ရာနှစ်၏ စာရင်းကို ချက်ချင်းပြောင်းလဲပြသပေးပါတယ်။
    *   Website စဖွင့်ဖွင့်ချင်း နောက်ဆုံးနှစ် (ဥပမာ: ၂၀၂၅) ၏ စာရင်းကို အလိုအလျောက်ပြသပေးပါတယ်။

*   **စာရင်းပြသခြင်းနှင့် အစဉ်စီခြင်း (Display & Sorting):**
    *   စာရင်းများကို "ကျောင်း" အလိုက်၊ ပြီးမှ "အတန်း" အလိုက်၊ ပြီးမှ "ခုံနံပါတ်" အလိုက် အလိုအလျောက် အစဉ်စီပြီး ဇယားထဲမှာ ပြသပေးပါတယ်။
    *   "Left" (ထွက်သွားပြီး) လို့ သတ်မှတ်ထားတဲ့ သံဃာတော်များကို စာရင်းထဲမှာ မီးခိုးရောင်နဲ့ စာကြောင်းခြစ်ပြီး ပြသပေးပါတယ်။

*   **ရှာဖွေခြင်းနှင့် စစ်ထုတ်ခြင်း (Search & Filtering):**
    *   **နာမည်ဖြင့် ရှာဖွေခြင်း:** နာမည်တစ်စိတ်တစ်ပိုင်းကို ရိုက်ထည့်လိုက်ရုံနဲ့ သက်ဆိုင်တဲ့စာရင်းကို ချက်ချင်းရှာဖွေပေးပါတယ်။
    *   **အခြေအနေ (Status) စစ်ထုတ်ခြင်း:** "လက်ရှိရှိသူများ"၊ "ထွက်သွားသူများ"၊ "အားလုံး" ဆိုပြီး ရွေးချယ်စစ်ထုတ်နိုင်ပါတယ်။
    *   **အတန်း/ကျောင်း စစ်ထုတ်ခြင်း:** Dropdown List ကနေ အတန်း သို့မဟုတ် ကျောင်းကို ရွေးချယ်ပြီး စာရင်းကို ကျဉ်းမြောင်းအောင် စစ်ထုတ်နိုင်ပါတယ်။ Dropdown ထဲမှာ လက်ရှိရှိနေတဲ့ သံဃာအရေအတွက်ကိုပါ တစ်ခါတည်း ပြသပေးပါတယ်။
    *   **ဘာသာရပ် စစ်ထုတ်ခြင်း:** ဘာသာရပ် အရေအတွက် သို့မဟုတ် တိကျတဲ့ ဘာသာရပ်တစ်ခုခုနဲ့ စစ်ထုတ်နိုင်ပါတယ်။
    *   **ရှင်းလင်းမည် (Clear Filters):** ခလုတ်တစ်ချက်နှိပ်ရုံနဲ့ Filter အားလုံးကို မူလအခြေအနေအတိုင်း ပြန်ထားပေးပါတယ်။

*   **Admin Panel (စီမံခန့်ခွဲမှုအပိုင်း):**
    *   **ဝင်ရောက်ခြင်း:** "Admin Panel" ခလုတ်ကိုနှိပ်ပြီး Password (`Ashin@135`) ကို မှန်ကန်စွာ ရိုက်ထည့်မှ ဝင်ရောက်နိုင်ပါတယ်။
    *   **ကျောင်းသားသစ် ထည့်ခြင်း၊ ပြင်ဆင်ခြင်း၊ ဖျက်ခြင်း:** လက်ရှိကြည့်ရှုနေသော ခုနှစ်၏ စာရင်းထဲသို့ ကျောင်းသားသစ်ထည့်ခြင်း၊ အချက်အလက်ပြင်ဆင်ခြင်း (ဥပမာ- Status ကို "Left" ပြောင်းခြင်း)၊ နှင့် စာရင်းမှ ဖယ်ရှားခြင်းတို့ကို ပြုလုပ်နိုင်ပါတယ်။
    *   **CSV ဖိုင်အသစ် ဒေါင်းလုဒ်ဆွဲခြင်း:** Admin Panel ထဲမှာ ပြင်ဆင်မှုတွေအားလုံးလုပ်ပြီးနောက်၊ "CSV ဖိုင်အသစ်ကို ဒေါင်းလုဒ်လုပ်ရန်" ခလုတ်ကိုနှိပ်ပြီး **လက်ရှိကြည့်ရှုနေသော ခုနှစ်အတွက်** နောက်ဆုံးအခြေအနေနှင့် ကိုက်ညီသော CSV ဖိုင်အသစ် (`students-2025.csv`) ကို ကိုယ့်စက်ထဲသို့ ဒေါင်းလုဒ်ဆွဲနိုင်ပါတယ်။

### **၄။ နောင်နှစ်အတွက် အချက်အလက်အသစ် ထည့်သွင်းနည်း (Step-by-Step Guide for Next Year)**

ဥပမာအနေဖြင့် **၂၀၂၇ ခုနှစ်** အတွက် စာရင်းအသစ်ကို ထပ်မံထည့်သွင်းလိုပါက အောက်ပါအဆင့် (၃) ဆင့်ကို လိုက်နာရုံဖြင့် အလွယ်တကူ ပြုလုပ်နိုင်ပါတယ်-

**အဆင့် ၁: `students-2027.csv` ဖိုင်အသစ် ဖန်တီးပါ**

*   သင့်ကွန်ပျူတာထဲမှာ `students-2025.csv` ဖိုင်ကို copy ကူးပြီး `students-2027.csv` လို့ နာမည်ပြောင်းလိုက်ပါ။
*   `students-2027.csv` ဖိုင်အသစ်ကို Microsoft Excel သို့မဟုတ် Google Sheets ဖြင့် ဖွင့်ပါ။ (စာလုံးမှန်အောင် Data > From Text/CSV မှ ဖွင့်ပါ)။
*   စာရင်းအဟောင်းများကို ဖျက်ပြီး ၂၀၂၇ ခုနှစ်အတွက် စာရင်းအသစ်များကို ကော်လံခေါင်းစဉ်များ (Name, Class, School...) အတိုင်း ထည့်သွင်းပါ။
*   **အရေးကြီး:** Class နဲ့ School ကော်လံတွေမှာ အရှေ့က အစဉ်လိုက်နံပါတ် (`၀၁-`, `၀၂-`) ကို မဖြစ်မနေ ထည့်ပေးပါ။ ဒါမှ Website က အစဉ်ကို မှန်အောင်စီနိုင်မှာပါ။ ဥပမာ - `၀၅-အကြီးတန်း`၊ `၀၂-လှ`။
*   ပြင်ဆင်ပြီးပါက CSV format ဖြင့် Save လုပ်ပါ။

**အဆင့် ၂: `index.html` ဖိုင်တွင် ခုနှစ်အသစ်ကို ထည့်သွင်းပါ**

*   `index.html` ဖိုင်ကို Text Editor (Notepad++, VS Code) ဖြင့် ဖွင့်ပါ။
*   အောက်ပါ `<select>` အပိုင်းကို ရှာပါ-
    ```html
    <select id="yearSelector" class="border p-2 rounded">
        <option value="2025" selected>၂၀၂၅</option>
        <option value="2026">၂၀၂၆</option>
    </select>
    ```
*   အဲဒီထဲမှာ ၂၀၂၇ အတွက် စာကြောင်းတစ်ကြောင်း အောက်ပါအတိုင်း ထပ်ထည့်လိုက်ပါ-
    ```html
    <select id="yearSelector" class="border p-2 rounded">
        <option value="2025">၂၀၂၅</option>
        <option value="2026">၂၀၂၆</option>
        <option value="2027" selected>၂၀၂၇</option> <!-- "selected" ကို နောက်ဆုံးနှစ်မှာ ထားပေးပါ -->
    </select>
    ```
    **အကြံပြုချက်:** `selected` ဆိုတဲ့ attribute ကို နောက်ဆုံးပေါ်စေချင်တဲ့ ခုနှစ်ရဲ့ `<option>` မှာ ထားပေးပါ။ ဒါဆို Website စဖွင့်ချင်း အဲဒီနှစ်ကို အရင်ဆုံးပြပါလိမ့်မယ်။
