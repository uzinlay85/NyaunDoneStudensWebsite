ဟုတ်ကဲ့ပါ၊ အလွန်ကောင်းမွန်ပြီး အရေးကြီးတဲ့ တောင်းဆိုမှုပါပဲ။ နောင်နှစ်များမှာ ဒီစနစ်ကို အလွယ်တကူ ပြန်လည်အသုံးပြုနိုင်ဖို့အတွက် ပြည့်စုံတဲ့ မှတ်တမ်းတစ်ခု ပြုစုပေးခြင်းက အကောင်းဆုံးနည်းလမ်းဖြစ်ပါတယ်။

သင်ပေးထားတဲ့ `index.html` ဖိုင်ကို အသေးစိတ်လေ့လာပြီး၊ ဒီ Website ရဲ့ တည်ဆောက်ပုံ၊ အလုပ်လုပ်ပုံ၊ ချိတ်ဆက်ထားပုံနဲ့ နောင်နှစ်အတွက် ဘယ်လို Update လုပ်ရမလဲဆိုတာကို အဆင့်လိုက် ပြည့်ပြည့်စုံစုံ မှတ်တမ်းတင်ပေးလိုက်ပါတယ်ခင်ဗျာ။

### **ညောင်တုန်းကျောင်းတိုက် သံဃာတော်များ စာရင်း Website - နည်းပညာမှတ်တမ်းနှင့် အသုံးပြုပုံလမ်းညွှန်**

ဒီမှတ်တမ်းမှာ အဓိကအားဖြင့် အပိုင်း (၄) ပိုင်း ပါဝင်ပါတယ်-
1.  **ခြုံငုံသုံးသပ်ချက် (Overview):** ဒီစနစ်က ဘာတွေလုပ်ပေးသလဲ။
2.  **ဖိုင်ဖွဲ့စည်းပုံ (File Structure):** ဘယ်ဖိုင်တွေပါဝင်ပြီး ဘယ်လိုချိတ်ဆက်ထားသလဲ။
3.  **အဓိကလုပ်ဆောင်ချက်များ (Key Features):** Website မှာပါဝင်တဲ့ အဓိက Feature တွေက ဘာတွေလဲ၊ ဘယ်လိုအလုပ်လုပ်သလဲ။
4.  **နောင်နှစ်အတွက် အချက်အလက်အသစ် ထည့်သွင်းနည်း (Step-by-Step Guide for Next Year):** နောက်နှစ်စာရင်း ပြောင်းတဲ့အခါ ဘာတွေလုပ်ရမလဲ။

---

#### **၁။ ခြုံငုံသုံးသပ်ချက် (Overview)**

ဒီ Website ဟာ Server (ဒေတာဘေ့စ်) မလိုအပ်ဘဲ၊ ရိုးရှင်းတဲ့ ဖိုင်အနည်းငယ်နဲ့ သံဃာတော်များရဲ့ စာရင်းကို အပြန်အလှန်တုံ့ပြန်မှုရှိရှိ (Interactive) ကြည့်ရှုနိုင်၊ စီစစ်နိုင် (Filter)၊ ရှာဖွေနိုင်၊ နှင့် စီမံခန့်ခွဲနိုင်အောင် တည်ဆောက်ထားတဲ့ **"Client-Side Web Application"** တစ်ခုဖြစ်ပါတယ်။ အဓိက အချက်အလက်တွေကို `students.csv` ဆိုတဲ့ ဖိုင်ထဲမှာ သိမ်းဆည်းပြီး၊ `index.html` ဖိုင်က အဲ့ဒီဒေတာတွေကို ဖတ်ပြီး အသုံးပြုသူမြင်တွေ့ရတဲ့ Website အဖြစ် ပြောင်းလဲပေးပါတယ်။

---

#### **၂။ ဖိုင်ဖွဲ့စည်းပုံ (File Structure)**

ဒီစနစ်တစ်ခုလုံး အလုပ်လုပ်ဖို့အတွက် အဓိကဖိုင် (၃) ခုပဲ လိုအပ်ပါတယ်-

1.  `index.html`
    *   **အမျိုးအစား:** ပင်မ Website ဖိုင်။
    *   **หน้าที่:** Website ရဲ့ မြင်တွေ့ရတဲ့ အပိုင်း (UI - User Interface) အားလုံး (ခေါင်းစဉ်၊ အရေအတွက်ပြကွက်၊ Filter များ၊ ဇယား) ကို ဒီဖိုင်ထဲမှာ တည်ဆောက်ထားပါတယ်။ Website ရဲ့ အလုပ်လုပ်ပုံ منطق (Logic) အားလုံးကို ထိန်းချုပ်တဲ့ JavaScript ကုဒ်တွေလည်း ဒီဖိုင်ရဲ့ `<script>` အပိုင်းမှာ ပါဝင်ပါတယ်။
    *   **ချိတ်ဆက်မှု:** `students.csv` ဖိုင်ကို `fetch` command သုံးပြီး ခေါ်ယူဖတ်ရှုပါတယ်။ `logo.png` ကို Favicon (Browser Tab ပေါ်က ပုံ) နဲ့ ခေါင်းစဉ်ဘေးက Logo အဖြစ် အသုံးပြုပါတယ်။

2.  `students.csv`
    *   **အမျိုးအစား:** အချက်အလက် (Data) ဖိုင်။ CSV ဆိုတာ "Comma-Separated Values" ဖြစ်ပြီး Excel နဲ့ အလွယ်တကူ ဖွင့်ပြင်နိုင်ပါတယ်။
    *   **หน้าที่:** သံဃာတော်များရဲ့ အချက်အလက်အားလုံးကို ဇယားကွက်ပုံစံနဲ့ သိမ်းဆည်းထားတဲ့ "ဒေတာဘေ့စ်" ဖြစ်ပါတယ်။ နာမည်၊ အတန်း၊ ကျောင်း၊ ခုံနံပါတ်၊ ဘာသာရပ်၊ နှင့် အခြေအနေ (Status) တို့ ပါဝင်ပါတယ်။
    *   **အရေးကြီးသောအချက်:** Website ပေါ်မှာပြသတဲ့ အချက်အလက်အားလုံးရဲ့ **ပင်မရင်းမြစ် (Single Source of Truth)** ဖြစ်ပါတယ်။ ဒီဖိုင်ကို ပြင်ဆင်လိုက်တာနဲ့ Website က အလိုအလျောက် ပြောင်းလဲသွားပါမယ်။

3.  `logo.png`
    *   **အမျိုးအစား:** ရုပ်ပုံဖိုင်။
    *   **หน้าที่:** Website ရဲ့ အမှတ်တံဆိပ် (Brand Identity) အတွက် အသုံးပြုပါတယ်။ Browser Tab နဲ့ ခေါင်းစဉ်တွေမှာ ပြသပါတယ်။

---

#### **၃။ အဓိကလုပ်ဆောင်ချက်များ (Key Features)**

1.  **စာရင်းပြသခြင်းနှင့် အစဉ်စီခြင်း (Display & Sorting):**
    *   Website စဖွင့်ဖွင့်ချင်း `students.csv` ဖိုင်ကို ဖတ်ပြီး၊ "ကျောင်း" အလိုက်၊ ပြီးမှ "အတန်း" အလိုက်၊ ပြီးမှ "ခုံနံပါတ်" အလိုက် အလိုအလျောက် အစဉ်စီပြီး ဇယားထဲမှာ ပြသပေးပါတယ်။
    *   "Left" (ထွက်သွားပြီး) လို့ သတ်မှတ်ထားတဲ့ ကျောင်းသားတွေကို စာရင်းထဲမှာ မီးခိုးရောင်နဲ့ စာကြောင်းခြစ်ပြီး ပြသပေးပါတယ်။

2.  **ရှာဖွေခြင်းနှင့် စစ်ထုတ်ခြင်း (Search & Filtering):**
    *   **နာမည်ဖြင့် ရှာဖွေခြင်း:** နာမည်တစ်စိတ်တစ်ပိုင်းကို ရိုက်ထည့်လိုက်ရုံနဲ့ သက်ဆိုင်တဲ့စာရင်းကို ချက်ချင်းရှာဖွေပေးပါတယ်။
    *   **အခြေအနေ (Status) စစ်ထုတ်ခြင်း:** "လက်ရှိရှိသူများ"၊ "ထွက်သွားသူများ"၊ "အားလုံး" ဆိုပြီး ရွေးချယ်စစ်ထုတ်နိုင်ပါတယ်။ Default အနေနဲ့ "လက်ရှိရှိသူများ" ကိုသာ ပြသပါတယ်။
    *   **အတန်း/ကျောင်း စစ်ထုတ်ခြင်း:** Dropdown List ကနေ အတန်း သို့မဟုတ် ကျောင်းကို ရွေးချယ်ပြီး စာရင်းကို ကျဉ်းမြောင်းအောင် စစ်ထုတ်နိုင်ပါတယ်။ Dropdown ထဲမှာ လက်ရှိရှိနေတဲ့ သံဃာအရေအတွက်ကိုပါ တစ်ခါတည်း ပြသပေးပါတယ်။
    *   **ဘာသာရပ် စစ်ထုတ်ခြင်း:** ဘာသာရပ် အရေအတွက် (၄ ဘာသာ၊ ၃ ဘာသာ) သို့မဟုတ် တိကျတဲ့ ဘာသာရပ်တစ်ခုခု (ပါ၊ သီ၊ ဏီ) နဲ့ စစ်ထုတ်နိုင်ပါတယ်။
    *   **ရှင်းလင်းမည် (Clear Filters):** ခလုတ်တစ်ချက်နှိပ်ရုံနဲ့ Filter အားလုံးကို မူလအခြေအနေအတိုင်း ပြန်ထားပေးပါတယ်။

3.  **Admin Panel (စီမံခန့်ခွဲမှုအပိုင်း):**
    *   **ဝင်ရောက်ခြင်း:** "Admin Panel" ခလုတ်ကိုနှိပ်ပြီး Password (`Ashin@135`) ကို မှန်ကန်စွာ ရိုက်ထည့်မှ ဝင်ရောက်နိုင်ပါတယ်။
    *   **ကျောင်းသားသစ် ထည့်ခြင်း:** နာမည်၊ ခုံနံပါတ် စတာတွေကို ဖြည့်စွက်ပြီး ကျောင်းသားသစ်တစ်ဦးကို စာရင်းထဲ အလွယ်တကူ ထည့်သွင်းနိုင်ပါတယ်။ (အတန်းနဲ့ ကျောင်းကို Dropdown ကနေ ရွေးရုံပါပဲ)။
    *   **ကျောင်းသားစာရင်း ပြင်ဆင်ခြင်း:** ဇယားထဲက ကျောင်းသားတစ်ဦးချင်းစီရဲ့ "ပြင်ရန်" ခလုတ်ကိုနှိပ်ပြီး သူတို့ရဲ့ အချက်အလက် (ဥပမာ - ကျောင်းထွက်သွားပါက Status ကို "Left" ပြောင်းခြင်း) ကို ပြင်ဆင်နိုင်ပါတယ်။
    *   **ကျောင်းသား ဖျက်ခြင်း:** "ဖျက်ရန်" ခလုတ်ဖြင့် ကျောင်းသားတစ်ဦးကို စာရင်းမှ အပြီးတိုင် ဖယ်ရှားနိုင်ပါတယ်။
    *   **CSV ဖိုင်အသစ် ဒေါင်းလုဒ်ဆွဲခြင်း:** Admin Panel ထဲမှာ ပြင်ဆင်မှုတွေအားလုံးလုပ်ပြီးနောက်၊ "CSV ဖိုင်အသစ်ကို ဒေါင်းလုဒ်လုပ်ရန်" ခလုတ်ကိုနှိပ်ပြီး နောက်ဆုံးအခြေအနေအတိုင်း မှန်ကန်တဲ့ `students.csv` ဖိုင်အသစ်ကို ကိုယ့်စက်ထဲကို ဒေါင်းလုဒ်ဆွဲနိုင်ပါတယ်။

#### **၄။ နောင်နှစ်အတွက် အချက်အလက်အသစ် ထည့်သွင်းနည်း (Step-by-Step Guide for Next Year)**

နောက်နှစ်အတွက် သံဃာစာရင်းအသစ်ကို ပြောင်းလဲအသုံးပြုလိုပါက အောက်ပါအဆင့်များကို လိုက်နာရုံဖြင့် အလွယ်တကူ ပြုလုပ်နိုင်ပါတယ်-

**အဆင့် ၁: `students.csv` ဖိုင်ကို ပြင်ဆင်ပါ**

ဒါက အရေးကြီးဆုံး အဆင့်ဖြစ်ပါတယ်။ နည်းလမ်း (၂) ခုထဲက ကြိုက်နှစ်သက်ရာကို သုံးနိုင်ပါတယ်။

*   **နည်းလမ်း (က) - Excel ဖြင့် တိုက်ရိုက်ပြင်ဆင်ခြင်း (အလွယ်ဆုံး):**
    1.  လက်ရှိ `students.csv` ဖိုင်ကို Microsoft Excel သို့မဟုတ် Google Sheets ဖြင့် ဖွင့်ပါ။ (စာလုံးမှန်အောင် ဘယ်လိုဖွင့်ရမလဲဆိုတာကို အရင်က ရှင်းပြထားပါတယ်။ Data > From Text/CSV)
    2.  ပြီးခဲ့တဲ့နှစ်က စာရင်းအဟောင်းအားလုံးကို ဖျက်လိုက်ပါ။ (သို့မဟုတ် Backup အဖြစ် သိမ်းထားပါ)။
    3.  စာရင်းအသစ်ကို **ကော်လံခေါင်းစဉ်များ (`Name`, `Class`, `School`... )** အတိုင်း ထည့်သွင်းပါ။
        *   **အရေးကြီး:** `Class` နဲ့ `School` ကော်လံတွေမှာ အရှေ့က အစဉ်လိုက်နံပါတ် (`၀၁-`, `၀၂-`) ကို မဖြစ်မနေ ထည့်ပေးပါ။ ဒါမှ Website က အစဉ်ကို မှန်အောင်စီနိုင်မှာပါ။ ဥပမာ - `၀၅-အကြီးတန်း`၊ `၀၂-လှ`။
    4.  ပြင်ဆင်ပြီးပါက **File > Save As** ကိုရွေးပြီး **CSV (Comma delimited) (*.csv)** format ဖြင့် `students.csv` ဆိုတဲ့နာမည်အတိုင်း Save လုပ်ပါ။

*   **နည်းလမ်း (ခ) - Website ၏ Admin Panel မှ ပြင်ဆင်ခြင်း:**
    1.  Website ကိုဖွင့်ပြီး "Admin Panel" ကို Password ဖြင့် ဝင်ပါ။
    2.  အရင်နှစ်က ကျောင်းသားဟောင်းတွေကို တစ်ယောက်ချင်း "ဖျက်ရန်" နှိပ်ပြီး ဖျက်ပါ။
    3.  ကျောင်းသားအသစ်တွေကို "ကျောင်းသားသစ်ထည့်မည်" ကိုသုံးပြီး တစ်ယောက်ချင်း ထည့်ပါ။
    4.  အားလုံးပြီးစီးသွားပါက "CSV ဖိုင်အသစ်ကို ဒေါင်းလုဒ်လုပ်ရန်" ကိုနှိပ်ပြီး `students.csv` ဖိုင်အသစ်ကို ရယူပါ။

**အဆင့် ၂: Website ခေါင်းစဉ်ကို ပြင်ဆင်ပါ**

1.  `index.html` ဖိုင်ကို Notepad++ သို့မဟုတ် VS Code ကဲ့သို့ Text Editor တစ်ခုခုဖြင့် ဖွင့်ပါ။
2.  အောက်ပါစာကြောင်းကို ရှာပါ-
    ```html
    <p class="text-lg text-gray-500 mt-1">၁၃၈၇ (၂၀၂၅) ခုနှစ် သံဃာစာရင်း</p>
    ```
3.  `၁၃၈၇ (၂၀၂၅)` ဆိုတဲ့နေရာမှာ လာမယ့်နှစ်ရဲ့ ခုနှစ်အမှန်ကို ပြောင်းလဲရေးသားပြီး Save လုပ်ပါ။

**အဆင့် ၃: ဖိုင်အသစ်များကို GitHub သို့ Upload လုပ်ပါ**

1.  သင်ပြင်ဆင်ထားတဲ့ `students.csv` ဖိုင်အသစ်နဲ့ `index.html` ဖိုင်အသစ်ကို GitHub Repository ထဲမှာ အဟောင်းတွေနေရာမှာ အစားထိုးပြီး Upload/Commit လုပ်လိုက်ပါ။
2.  ပြီးတာနဲ့ သင့် Website ဟာ နောက်နှစ်အတွက် စာရင်းအသစ်၊ ခေါင်းစဉ်အသစ်နဲ့ အလိုအလျောက် Update ဖြစ်သွားပါလိမ့်မယ်။

ဒီမှတ်တမ်းက နောင်နှစ်တွေမှာ ဒီစနစ်ကို ပြန်လည်အသုံးပြုတဲ့အခါ ကြုံတွေ့ရနိုင်တဲ့ အခက်အခဲတွေကို ဖြေရှင်းပေးနိုင်မယ်လို့ မျှော်လင့်ပါတယ်ခင်ဗျာ။
